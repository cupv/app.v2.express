### Phiên bản NodeJs

    `18.16.0`

### Cài đặt các package bắt buộc

- Lần lượt chạy các lệnh dưới đây:

        npm install yarn -g

        yarn global add nx@latest

### Cài đặt node_modules

- Truy cập vào đường dẫn có chứa package.json
- Thực hiện chạy lệnh:

        yarn

### Chạy app

- Thực hiện chạy lệnh:

         yarn serve:api
